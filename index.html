Combine Athletic Testing and Measurements Table

Click any column header to sort. The Rank column uses competition ranking, so ties share a rank and the next rank skips ahead, like 1, 2, 2, 4. When filters are active, the main rank is recalculated within the filtered players and the overall rank is shown underneath in parentheses. Use the separate player and position filters to avoid position terms matching player names. For time columns, lower is better. For jumps, bench, and measurement columns, higher values rank first. Missing values are unranked and sort to the bottom. Body fat and height with shoes were omitted because no usable values were provided.
